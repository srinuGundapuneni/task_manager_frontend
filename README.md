
Employee Task Tracker

A beautiful, modern, and fully responsive Employee Task Management application built with React, Tailwind CSS, and Framer Motion. This application allows you to view employees, manage their tasks, filter by status, and track project progress with an interactive dashboard.

Features

 Core Features
- Interactive Dashboard with animated statistics cards
- Employee Management with expandable task lists
- Task Filtering by status (All, Pending, In Progress, Completed)
- Add New Tasks with modal form and immediate UI updates
- Status Management with color-coded badges and quick status toggle
- Delete Tasks with smooth animations
- LocalStorage Persistence** - data survives page refreshes

 UI/UX Highlights
-Smooth Animations powered by Framer Motion
-Beautiful Gradients and glassmorphism effects
-Fully Responsive design (mobile, tablet, desktop)
-Interactive Hover Effects and micro-interactions
- Animated Counter for dashboard statistics
- Floating Action Button for mobile devices
- Modern UI Components with Tailwind CSS

 Tech Stack

- Frontend Framework:React 18.2
- Styling: Tailwind CSS 3.3
- Animations: Framer Motion 10.16
- Icons:Lucide React
- Build Tool: Vite 5.0
- State Management: React Hooks (useState, useEffect)
- Data Persistence: LocalStorage API
